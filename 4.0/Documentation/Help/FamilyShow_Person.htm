    Dates

    To input approximate dates, left click the date field label to cycle the date descriptor to ABT, AFT or BEF for About, After or Before. Right click cycles the other way. This functionality is available in the Edit Details, Edit More Details and Relationships panels.

    Citations and Sources

    Each event field (Birth, Death, Occupation, Education, Religion, Burial, Cremation, Marriage and Divorce) can have a single citation. A valid Citation has details and states the Source reference.

    Special Fields

    These can be accessed from the Photos & Story view or the Attachments panel.

  • Attachments: A person can have a series of files attached to them.
  • Photos: A person can have a series of photos attached to them. Users can specify a primary image which is displayed a person's tooltip.
  • Story: A Rich Text File (*.rtf) note can be used to store other important information about people. This data is also stored in a Unicode text field.
