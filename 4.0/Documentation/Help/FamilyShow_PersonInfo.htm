Photos & Story

This screen is accessed using the Photos & Story button in the Details panel. It displays the story and photos of the current person. Stories and photos can also be viewed off the Tools menu. Here all stories and photos in a tree can be viewed.



Photos

Photos can be added by dropping them into the Photo Box or by using the Add menu. Images added to another person can be linked to the current person using the "Link existing image" option.

Additional options for setting the primary image, deleting images, and viewing images can be accessed in the Tools menu.


Fig 1. Photo options

Story

A story can be added to a person to record additional information which is not stored in the supported fields. The Story is in Rich Text Format (*.rtf) and can be printed via the Print button. Spell check is available via the right click menu in the editor.
