Attachments

Files can be added to a person in the Attachments panel. The following file formats are supported: *.doc, *.docx, *.xls, *.xlsx, *.ppt, *.pptx, *.odp, *.odt, *.ods, *.xps , *.htm, *.html, *.pdf, *.txt, *.kml, *.kmz, *.jpeg, *.jpg, *.png, *.tif, *.tiff, *.gif, *.rtf.


Fig 1. The Attachments panel Viewing all attachments

Attachments can also be viewed off the Tools menu. Here all attachments in a tree can be viewed.
