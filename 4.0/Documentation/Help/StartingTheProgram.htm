Starting the program

To start Family.Show, navigate to its entry in Start>Programs>Family.Show>Family.Show.

If you are using Windows 7, you can take advantage of the Family.Show Jump List which allows you to access common tasks such as "Start a new family tree...", "Import GEDCOM " and "Open...".
