HTML Export

This option can be used to create tabulated reports of family data in the XHTML 1.0 Transitional specification*. These can be viewed in any modern web browser. Where present, notes are exported in a collapsed field. The note can be toggled on and off by clicking the [Note] link. This functionality uses a JavaScript. If JavaScript is disabled, notes will be visible by default. Citation , Source and Repository information can also be included if this is desired. HTML Export supports privacy.



*Note that the report structure is to the standard. However, if data fields contain extended characters etc, the standardisation may be broken.
